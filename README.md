# Resume-Interview-Coach
Help job seekers prepare with feedback and mock interviews.

This project is an AI-powered career preparation assistant designed to help job seekers improve their resumes and interview performance. It uses a Resume Optimizer Agent to suggest stronger resume content and tailoring improvements, a Behavioral Interview Agent to generate mock interview questions and review answers, and a Role-Fit Analyzer Agent to evaluate how well a candidate matches a target role. These agents work together to support end-to-end job preparation, while the system stores resume reviews, interview responses, and feedback history so users can track their improvement over time.
